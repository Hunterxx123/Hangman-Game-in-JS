# Hangman-Game-in-JS
A Hangman Game, written in Javascript powered by NodeJS

Hangman Game:

Tests on Geography words
There will be 3 hints given :

1) Show Vowels
2) Show Definition
3) Allow user to score and move on to the next word

The player scores when he or she guesses the word correctly

The user can only make 2 mistakes out of guessing 10 words, upon making the third mistake, he or she loses the game
